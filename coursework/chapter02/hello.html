Hello! You've found the coursework folders!

Most chapters have coding exercises for you to complete. Some are "Debugging" exercises where you get to find the problem and correct it. Some are "Code Modification" exercises where you must add or modify code for some new purpose. And some are "Code Completion" exercises where you must complete the code to meet the requirements. This approach will help you build your skills one step at a time.

The files containing the code for these exercises are located in the appropriate chapter folder under the coursework folder. You can just open each file, complete the exercise, save the file, and then test your work by viewing the file with your Web browser. And repeat as needed until you are ready to submit your work.

NOTE: You might want to make a copy of the coursework folder in case you make mistakes and want to retrieve the original files.
